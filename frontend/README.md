# Inventory Management System - Frontend

## Tech Stack
- React 18 + Vite
- Tailwind CSS v4
- React Router v6
- Axios
- Recharts
- React Hook Form + Zod
- React Hot Toast
- Lucide React

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Copy `.env.example` to `.env` and adjust the API base URL if needed.
3. Start the dev server:
   ```bash
   npm run dev
   ```
